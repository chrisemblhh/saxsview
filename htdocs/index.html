saxsview

libsaxsdocument and saxsview; read, convert and view 1D and 2D-files related to Small Angle X-ray Scattering (SAXS).

saxsview

saxsview is intended as an day-to-day plotting facility for SAXS-data, as well as a convenient tool to quickly generate plots ready for publication.

Basic plotting of 1D-curves is available (.dat, .int, .fit and .fir formats of the ATSAS package, as well as the cansas XML format v1.0), so is the display of 2D scattering images in Crystallographic Binary Format (CBF, using CBFlib) and the display of 32-bit TIFFs as written by PILATUS detectors.

The program is based on Qt - A cross-platform application and UI framework and Qwt - Qt Plotting Widgets for Technical Applications.

Suggestions, bug reports and code contributions are highly welcome!

libsaxsdocument

libsaxsdocument is the reading and writing workhorse of saxsview. It is meant to allow for easy integration of new 1D and 2D-file formats. Further, if formats that hold similar information implement reading and writing hooks, it becomes very easy to convert between these respective formats, eventually simplifying data-transfer between programs.

For later releases, support for the MAR345 file format is planned.

External Dependencies

External dependencies so far; version numbers indicate minimum requirements, earlier versions may work, but are not guaranteed to work. Later version are generally assumed to work:

  • CMake-2.6 (build system)
  • Qt-4.4 (UI framework)
  • gcc-4.3 (C, C++ and Fortran compilers)
Optional dependencies which are used if found on the build system:
  • LibTIFF (support for 32-bit TIFF images)
  • LibXml2 (support for cansas XML file format)

Dependecies less commonly found (Qwt, CBFlib) on average systems come together with the saxsview sources.

Building saxsview

The commands listed here are for a bash-shell on Linux. Similar commands apply for Windows or MacOSX builds: 

  $> tar xfz saxsview-0.1.0.tar.gz
  $> cd saxsview-0.1.0
  $> mkdir build && build
  $> cmake .. -DCMAKE_INSTALL_PREFIX=/where/to/install
  $> make
  $> make test      # optional, check that libsaxsdocument works
  $> make install

As is, it was successfully compiled and used on Linux (i686-pc-linux-gnu) and Windows (i686-unknown-mingw), it is to expect that it works on Mac as well.
