saxsview

libsaxsdocument and saxsview; read, convert and view 1D and 2D-files related to Small Angle X-ray Scattering (SAXS).

saxsview

saxsview is intended as an day-to-day plotting facility for SAXS-data, as well as a convenient tool to quickly generate plots ready for publication.

Basic plotting of 1D-curves is available (.dat, .int, .fit/.fir formats of the ATSAS package), so is the display of 2D scattering images in Crystallographic Binary Format (CBF, using CBFlib).

The program is based on Qt - A cross-platform application and UI framework and Qwt - Qt Plotting Widgets for Technical Applications.

As is, it was successfully compiled and used on Linux (i686-pc-linux-gnu) and Windows (i686-unknown-mingw), it is to expect that it works on Mac as well.

libsaxsdocument

libsaxsdocument is the reading and writing workhorse of saxsview. It meant to allow for easy integration of new 1D and 2D-file formats. Further, if formats that hold similar information implement reading and writing hooks, it becomes very easy to convert between these respective formats. Eventually simplifying data-transfer between programs.

For later releases, support for TIFF32 (as written by PILATUS detectors) and the MAR345 file formats are planned.
